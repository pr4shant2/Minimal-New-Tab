# 🌌 Minimal New Tab Lite

A beautifully crafted, ultra-lightweight, and brutally honest "New Tab" extension for Chrome/Edge. Built with zero dependencies, 100% vanilla JavaScript, and optimized for absolutely zero unnecessary CPU/GPU overhead.

## ✨ Features
- **Savage Daily Quotes**: 110+ highly curated, brutally honest, and philosophical quotes (in Gujarati) that cycle systematically to wake you up to reality.
- **Dynamic Aurora Mesh Gradient**: A stunning, high-quality, pure CSS mesh gradient background that costs 0% extra CPU/GPU load.
- **Live Auto-Geocoded Weather**: Powered by the Open-Meteo API. Just type your city name in settings, and it automatically finds the exact coordinates to fetch real-time weather.
- **Year Progress Bar**: A sleek, drift-free visual indicator of how fast the year is passing by.
- **Glassmorphism UI**: Beautifully blurred, aesthetic widgets that seamlessly adapt to Dark & Light modes.
- **In-Page Settings Modal**: No need to visit the extension options page. Configure your Name and City directly from the New Tab page.
- **Privacy First**: No tracking, no heavy analytics, and zero bloated libraries. Your data stays in your browser's local storage.

## 🚀 Installation (Developer Mode)
1. Download or clone this repository.
2. Open your Chromium-based browser (Chrome, Edge, Brave).
3. Navigate to `chrome://extensions/` (or `edge://extensions/`).
4. Enable **Developer mode** in the top right corner.
5. Click **Load unpacked** and select the folder containing these files.
6. Open a new tab and enjoy!

## ⚙️ Configuration
- Click the **⚙️ Gear Icon** in the bottom right corner of the New Tab page to open the Settings Modal.
- Enter your Name for a personalized greeting.
- Enter your City Name to automatically fetch and display local weather data.

## 🛠️ Built With
- Pure HTML5 / CSS3
- Vanilla JavaScript (ES6+)
- [Open-Meteo API](https://open-meteo.com/) (Weather & Geocoding)
